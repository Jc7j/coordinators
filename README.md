# Coordinators

A software studio. We build and operate digital products, and partner with technology teams on the hard parts.

This is the studio's marketing site — Next.js (App Router), Tailwind v4, light/dark themes.

## Develop

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Structure

- `app/` — routes, layout, global styles + theme tokens
- `components/` — UI (theme toggle)
- `lib/projects.ts` — work list; add a project = add an entry
